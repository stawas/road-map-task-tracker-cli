# road-map-task-tracker-cli

## How to run

cago run

## commands

### Adding a new task
add "Buy groceries"
### Output: Task added!

### Updating and deleting tasks
update 1 "Buy groceries and cook dinner"
delete 1

### Marking a task as in progress, done or todo
mark-in-progress 1
mark-done 1
mark-todo 1

### Listing all tasks
list

### Listing tasks by status
list done
list todo
list in-progress
